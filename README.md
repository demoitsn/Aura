# Aura — Personal Productivity Dashboard

A minimal, beautiful productivity app with:
- ⏱ **Focus Timer** — Pomodoro technique with session logging
- ✓ **Tasks** — Priority-based task manager with filters
- 🔥 **Habits** — Daily habit tracker with streaks & 7-day history
- ◈ **Insights** — Stats, charts, and progress overview

All data is saved locally in your browser via localStorage.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```
